### r_intro_eph
- **URL:** https://github.com/pablotis/r_intro_eph
- **Descripción:** Introducción a R para Ciencias Sociales (versión anterior)
- **Contenido relevante:**
  - `0a_diapositivas/01a_r_intro_eph.Rmd` - Presentación sobre la EPH
  - `0a_diapositivas/01b_r_intro_presentacion.Rmd` - Intro a R

  - `0a_diapositivas/03_visualizacion.Rmd` - Visualización con ggplot2

### Uso
Para clonar localmente (no se sube al repo):
```bash
mkdir -p _referencias
cd _referencias
git clone https://github.com/pablotis/r_intro_eph.git
```
